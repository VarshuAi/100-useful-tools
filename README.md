<!-- ============================== SETUP ============================== -->

<h2>
<img src="https://media.giphy.com/media/LnQjpWaON8nhr21vNW/giphy.gif" width="28">
<samp>&nbsp;рџЊі SEED SOWING (Setup)</samp>
</h2>

### 1. Clone the Repository
```bash
git clone https://github.com/VarshuAi/100-useful-tools.git
cd 100-useful-tools
```

### 2. Install Dependencies
```bash
pip install -r requirements.txt
```

### 3. Launch the Interactive Menu
```bash
python runner.py
```

### 4. Or Run Any Tool Directly
```bash
python file_media/01_bulk_rename.py
python web_network/34_weather_cli.py
python study_exam/86_neet_countdown.py
```

### 5. Set Up Gemini API (for AI tools 61-90)
```bash
# Windows
set GEMINI_API_KEY=your_api_key_here

# Linux/Mac
export GEMINI_API_KEY=your_api_key_here
```
Get your free API key at [aistudio.google.com](https://aistudio.google.com)
